# React-comp-boiler
**DOES NOT SUPPORT FILE NAV**
Basic componant generater.

## Usage

`git clone git@github.com:Hobatron/react-comp-boiler.git`
`cd react-comp-boiler`
`npm i -g`
While in src:
`new-comp (YourComp) [-d (destination).js]`

## Requirments

File structure:
```tree
src
├──App.js
├──(destination).js
├──components
|   ├──YourComponents
```
